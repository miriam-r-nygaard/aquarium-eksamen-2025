<h2>aquarium-eksamen-2025</h2>

<h1>Eksamens opgave - Projekt i ExD "Akvarie" </h1>
<p>Helt overordent er der blevet fjernet en masse kode, fra tidligere forladte koncepter og lavet en mere koncentreret struktur med flere kommentarer og mere naturlig rækkefølge.</p>
<h3>Denne fil er blevet optimeret med følgende:</h3>
<h4>Assets</h4>
<ol>
<li><p>Rettet filnavne til små bogstaver</p></li>
<li><p>Lagt få ekstra SVG'er og lydfiler ind</p></li>
</ol>
<h4>HTML til Landingpage "index.html":</h4>
<ol>
<li><p>Optimeret knapper til relative stier</p></li>
<li><p>Indsat logo og nye paragraffer </p></li>
<li><p>Indsat mini-ikoner til knapper</p></li>
</ol>
<h4>CSS til Landingpage "landingpage.css":</h4>
<ol>
<li><p>CSS til knapper</p></li>
<li><p>Float animation til karakterene</p></li>
<li><p>Opdateret baggrunden</p></li>
</ol>
<h4> JavaScript til Landingpage "landing.js":</h4>
<ol>
<li><p>Opdateret toggle af pindsvinefisken "Nutte"</p></li>
<li><p>Lavet toggle funktion til blæksprutten "Sprutte"</p></li>
<li><p>Relative stier til knapper</p></li>
</ol>
<h4>HTML til subpage "video.html"</h4>
<ol>
<li><p>Opdaterede knapper</p></li>
<li><p>Indsat SVG'er med class "usynlig" for at kunne klikke mere optimalt på elementerne</p></li>
<li><p>Indsat et par ekstra SVG'er med nye fisk</p></li>
<li><p>Indsat et div til Overlay</p></li>
<li><p>Indsat logo</p></li>
</ol>
<h4>CSS til subpage "video.css"</h4>
<ol>
<li><p>Opdateret CSS til knapper</p></li>
<li><p>Omdannet px til responsive values så som vw, vh og em</p></li>
<li><p>Forsøgt responsivt design (virker i Browser, men ikke på iPad)</p></li>
<li><p>CSS til Overlay</p></li>
</ol>
<h4>JavaScript til subpage "video.js"</h4>
<ol>
<li><p>Opdateret funktion til knapper (nu med relativ sti på hjem knap)</p></li>
<li><p>Lavet array til alle fiskene</p></li>
<li><p>Lavet funktion til overlay med HTML skrevet i JavaScript</p></li>
<li><p>Samlet alle lydfiler i én const</p></li>
<li><p>Lavet lille overlay med infolyd, der afspilles, når man åbner subpagen</p></li>
<li><p>Opdateret toggle af pindsvinefisken "Nutte" og lavet en ekstra til Blæksprutten "Sprutte"</p></li>
</ol>
